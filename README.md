# BI Dom Frontend

A Business Intelligence (BI) frontend dashboard built to display powerful analytics and charts seamlessly.

## Key Features
- **Data Visualization**: Rich, interactive charts and graphs to visualize BI data efficiently.
- **Fast Data Fetching**: Robust state management and server state caching with TanStack React Query.
- **Smooth Animations**: Engaging UI transitions powered by Framer Motion.
- **Responsive Dashboard**: Beautiful and responsive layouts tailored for analytics.
- **Toast Notifications**: Interactive user feedback with Sonner.

## Tech Stack
- **Framework**: Next.js 16, React 19
- **Data Fetching**: `@tanstack/react-query`, Axios
- **Charts**: `chart.js`, `react-chartjs-2`
- **Animations**: Framer Motion
- **Utilities**: `js-cookie`, `lucide-react`, Tailwind CSS
- **Notifications**: Sonner
